Multi-half-bridge  3.0.2
This shield provides 12 half-bridges which can drive up to 6 independent (+5 cascaded) bidirectional motors consuming up to 0.9 Ampere each
multi-half-bridge

This code example demonstrate the basic usage of the Infineon TLE94112 multi-half-bridge

  • It will run the motor in forward and backward direction if the motor
  • is able to run in both directions. Otherwise it will only run in on direction.
  • Take this example as a first test when starting with this shield.
